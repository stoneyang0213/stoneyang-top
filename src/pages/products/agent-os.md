你的员工是 AI(Claude Code / skills / workflows),这里是他们协同工作的 OS。

## 为什么需要

一人公司最大的陷阱不是能力不够,是"一人身兼数职,精力被碎片化拖垮"。Agent OS 把一人公司重构成 5 大舱室,每个舱室只专注一件事。

## 5 大舱室架构

- **00-Kernel 系统内核** — 配置 / 规则 / 身份认知
- **01-Sensors 感知输入舱** — Inbox / 市场雷达 / 灵感冷藏
- **02-Brain 决策中枢** — 战略 / 规划 / 产品矩阵
- **03-Actuators 研发引擎** — IPD 研发 / 技能源码库
- **04-Memory 交付与输出** — IPMS 营销 / 变现资产

方法论来源:华为 BLM(业务领先模型)+ IPD(集成产品开发)+ IPMS(集成营销),翻译成一人公司可执行的版本。

## 三条核心行动原则

1. **过滤 > 收集** — 宁可错杀一千,不收无效垃圾
2. **行动 > 计划** — 粗糙 MVP 跑通一圈,再优化
3. **结果 > 感受** — 核心完美,非核心 60 分
